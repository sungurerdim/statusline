# Code of Conduct

This project is a small, focused tool maintained in the open. The goal is to
keep contribution friction low while keeping interactions respectful.

## Expected behavior

- Be respectful and constructive in issues, discussions, and pull requests.
- Focus feedback on the code and the problem, not the person.
- Assume good faith — most disagreements are about tradeoffs, not intent.

## Unacceptable behavior

- Harassment, personal attacks, or discriminatory language or jokes.
- Publishing others' private information without consent.
- Sustained disruption of issues, discussions, or pull requests.

## Scope

Applies to all project spaces: the issue tracker, pull requests, discussions,
and any other official communication channel for this repository.

## Enforcement

Report unacceptable behavior by opening a private [GitHub Security
Advisory](https://github.com/sungurerdim/statusline/security/advisories/new)
or contacting the maintainer directly through their GitHub profile. Reports
are handled confidentially. Maintainers may edit, remove, or reject
contributions and comments that violate this Code of Conduct, and may block
repeat offenders.
